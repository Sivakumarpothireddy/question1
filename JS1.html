JAVASCRIPT(JS)

How to attach JS file to HTML file?

  • To link js file to html use "script" tag inside body using "src" attribute.
  • Now write javascipt code in js file.
  • For a developer it is important to know about console. it is message box which shows where the errors was, or any hidden text only for developersto know..
  • To go to console: do inspect the webpage and then change elements tab to console tab.
NOTE:Javascipt file must be with ".js" extension.
